NaYaBoh - rasend schnelles Internet!

(English)

NaYaBoh ist ein Internet Proxy und Gateway mit Fokus auf eine einfache Benutzung.

NaYaBoh ist quelloffene Software (opensource) und deshalb gratis.

NaYaBoh verbessert eine teure oder langsame Anbindung ans Internet - speziell wenn mehrere Rechner die Anbindung nutzen. Ein paar Beispiele wo NaYaBoh Sinn macht:

  • Internet-Café in einem Entwicklungsland
  • Schule in einem abgelegenen Dorf
  • Büro mit langsamer Internetanbindung

NaYaBoh ist keine Wunderkiste. Es benutzt unter Experten altbekannte Technologien. NaYaBoh erlaubt dir von dem Expertenwissen zu profitieren ohne selber solch ein Experte zu sein. Wenn du NaYaBoh installierst, wird es versuchen eine gute Konfiguration zu finden und einfach still und leise seiner Arbeit nachzugehen. Um dir zu zeigen wie NaYaBoh dir hilft, hier eine kurze Auflistung:

  • Zwischenspeichern von bereits besuchten Webseiten, für schnelleren Zugriff nach dem ersten Aufruf
  • Blocken von Werbung
  • Zwischenspeicherung der Namensauflösung (DNS-Anfragen)
  • Automatische Netzwerkkonfiguration aller Rechner im Verbund
  • Kontrolle über die Anbindung einzelner Rechner möglich
  • Virus befallene Rechner können u.U. erkannt werden

Genug der langen Rede. Schauen wir uns mal dieses Bild an:

Installation in einem Informatik-Center in Buea, Kamerun. Etwa 30 Windows Rechner online gleichzeitig.

Auf der Y-Achse ist die Zeit abgebildet. Die zwei Graphen zeigen an wie lange das laden von Webseiten benötigte. Die rote Linie zeigt die normale Internet-Anbindung, die grüne Linie zeigt die Anbindung über NaYaBoh. Du siehst, das Internet mit NaYaBoh ist bedeutend schneller - die Linie liegt viel tiefer. Wenn du NaYaBoh installiert hast kannst du eine solche grafische Messung mit einem simplen Klick auch selber erstellen. So siehst du selber wie viel es dir bringt.

Entstehungsgeschichte

Ich fing damit an NaYaBoh zu entwickeln als ich in Kamerun war. Also habe ich das Projekt in einer lokalen Sprache benannt, Pidgin. In diesem Dialekt heisst "na ya boh" so viel wie "das ist dein Freund". Sicherlich ein treffender Name für dieses Stück Software!
